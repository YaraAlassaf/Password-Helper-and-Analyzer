# Password Helper and Analyzer

A Java-based application that helps users create strong passwords and evaluate their security based on common password strength criteria.

---

## Project Overview

This project was developed to improve password security by allowing users to:

- Generate secure random passwords
- Analyze password strength
- Provide recommendations for creating stronger passwords

---

## Features

- Secure password generation
- Password strength analysis
- Security recommendations
- User-friendly console interaction

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)

---

## Project Structure

The project consists of the following Java classes:

- Main.java
- PasswordHelper.java
- PasswordAnalyzer.java
